Xibalba

Creators
  • Dominic Szablewski
Tools
  • ImpactJS
Source
  • GitHub

Xibalba is a retro pixel-chic throwback to FPS classics of the 1990’s like Doom and Marathon. As creator Dominic Szablewski says in his Making of Xibalba post, “the game was built on top of my HTML5 Game Engine Impact and is entirely written in JavaScript.” In late 2014 Dominic added WebVR support for Xibalba, enabling uers to run and gun through pixel hallways via the WebVR and Gamepad APIs. Learn more about Dominic’s work and ImpactJS at phoboslab.org
