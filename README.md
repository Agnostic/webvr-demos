# webvr-demos

WebVR Demos.


## Installation

### GitHub

    # Clone the git repository from GitHub
    git clone git@github.com:MozVR/webvr-demos.git

    # Open the working directory
    cd webvr-demos

    # Install the Node dependencies
    npm install


## Development

To start the server:

    npm start

If you wish to serve the site from a different port:

    PORT=8000 npm start
